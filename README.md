# TerraNova Addon Wiki

## Setup

1. Prerequisites

- Install Node.js 20+.
- Optional: install pnpm or yarn if you prefer, but the examples below use npm.

2. Install dependencies

```bash
npm install
```

3. Run the dev server

```bash
npm run dev
```
Then open the URL shown in the terminal (defaults to http://localhost:4321).

4. Build for production

```bash
npm run build
```

5. Preview the production build locally

```bash
npm run preview
```
